layer.loading(obj) Object

该方法用法与layer.msg()非常类似，差别在于此处没有during来控制显示的时间，即用户需要在自己的加载动作完成后自行调用layer.close()来关闭(只有loading刻可以调用该方法关闭)。

            layer.loading({
                css: {
                    width: '100px',
                    backgroundPosition: 'center 16px',
                    padding: '60px 0 0 0',
                    fontSize: '12px'
                },
                text: '加载中'，
                loadingImg: ‘’
            })
            
            document.getElementById('close').addEventListener('click', function() {
                layer.close();
            });
1. text: string(需要)

该选项定义了所要显示的文字信息

            text: 'hello world'
2. css: object(可选)

该选项用于设置loading的样式，目前支持的参数有：width,height,lineHeight(如果传入连字符属性请以驼峰方式命名) ，backgroundColor,backgroundImage,backgroundSize,backgroundPosition,fontSize,padding,top,color,borderRadius,transition(过渡动画)，boxShadow,overflow。

            css: {
                top: '20%',
                fontSize: '14px'
                ...
            }
3. loadingImg: String(可选)

该属性用于定义loading的图片，图片的默认大小为30*30用户可在backgroundSize选项对图片大小进行设置

            loadingImg: '__ROOT__/public...'
