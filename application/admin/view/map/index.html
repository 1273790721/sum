new map(obj) Object

Map的使用非常简单，简单的一个调用即可实现最大众化的地图需求，因为百度api已经提供了相当全面的接口，因此如果想要更多自定义的东西请参阅http://lbsyun.baidu.com/index.php?title=jspopular/guide/introduction

注意：在此之前需要先引入百度地图的js文件 v2.0版本的引用方式：src=http://api.map.baidu.com/api?v=2.0&ak=您的密钥

            var map = new map({
                el: '#container',   
                point: [114.438814, 30.510813],
                zoom: 18,
                wheelZoom: true,
                markers: [{
                    lng: 114.429364,
                    lat: 30.548981
                }, {
                    lng: 114.524,
                    lat: 30.598
                }],
                // markerImg: '__PUBLIC__/images/friends.png',
                control: true,
                infoWindow: {
                    title: '武汉必腾移动科技有限责任公司',
                    width: 240,
                    height: 80,
                    content: '地址：武汉市东湖开发区珞瑜东路佳园路慧谷时空大厦604室'
                },
                showLocation: true
            });
1. showLocation: boolean(开发阶段用)

该属性是为了开发期间想要获取某位置的精确坐标而存在，使用者只需要点击该位置便可从控制台读取相关的坐标信息。

            showLocation: true
2. el: String(需要)

该参数需要用户来指定地图容器，需要id选择器来标识。

            el: ’#container’
3. point: Array(可选)

该参数用于指定当前地图的中心点，如果不传入，则会根据用户所在市的中心为中心点（经度纬度的获取方法请参照showLocation说明）。

            point: [114.438814, 30.510813]
4. zoom: Number(需要)

该参数用于设置地图的缩放级别，根据需要建议取值在20以下。

            zoom: 18
5. wheelZoom：Boolean(可选)

该参数用于设置用户是否可以通过鼠标滚轮缩放地图。

            wheelZoom: true
6. markers: Array(可选)

该参数用于指定地图范围内的标记点（经度纬度的获取方法请参照showLocation说明）。

            markers: [{
            lng: 114.429364,
            lat: 30.548981
        }, {
            lng: 114.524,
            lat: 30.598
        }]
7. markerImg: String(可选)

该参数用于指定标记的图片。

            markerImg: '/public/images/friends.png'
8. control: Boolean (可选)

该属性用于确定地图控制控件的显示隐藏。

            control: true
9. infoWindow：Object(可选)

该属性为当前地图中心点指定一个名片如下图。

            infoWindow: {
                title: '武汉必腾移动科技有限责任公司',    //  指定名片的标题
                width: 240,                             //  指定名片的宽高
                height: 80,
                content: '地址：武汉市东湖开发区珞瑜东路佳园路慧谷时空大厦604室'   //  指定名片的详细内容
            }
